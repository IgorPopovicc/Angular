  • Možete preuzeti Node.js sa zvanične Node.js stranice, a npm će biti automatski instaliran zajedno sa Node.js.

Verzije alatki

Važno je da održavate kompatibilnost između različitih verzija alatki kako biste osigurali da svi komponenti i delovi vašeg razvojnog okruženja rade zajedno bez problema. To znači da trebate pažljivo birati i ažurirati verzije različitih alatki, kao što su programski jezik, biblioteke, okviri i dodaci, kako bi se izbegli konflikti i nespojivosti. Kada koristite usklađene verzije alatki, povećava se stabilnost vašeg razvojnog procesa i smanjuje verovatnoća grešaka i problema prilikom razvoja i izgradnje vaših aplikacija.

  • Node.js i npm: Preporučuje se korišćenje LTS (Long-Term Support) verzije Node.js jer obezbeđuje stabilnost. npm će biti automatski instaliran uz Node.js

  • Angular CLI: Uvek koristite najnoviju stabilnu verziju Angular CLI, koju možete instalirati komandom npm install -g @angular/cli

  • TypeScript (ts): Angular se zasniva na TypeScript-u. Većina projekata koristi najnoviju stabilnu verziju TypeScript-a. U konkretnom projektu, verziju TypeScript-a možete pronaći u package.json datoteci u polju devDependencies
